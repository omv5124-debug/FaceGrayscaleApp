# Face Grayscale App

An Android application that detects faces using the front camera and converts the display to grayscale when a face is detected.

## Features
- ✅ Real-time face detection using front camera
- ✅ Grayscale screen conversion on face detection
- ✅ Foreground service for background operation
- ✅ Permission handling for camera access
- ✅ Auto-build with GitHub Actions

## Requirements
- Android SDK 24 (API Level 24) or higher
- Android 8.0 (API 26) or higher for runtime
- Camera permission
- JDK 17
